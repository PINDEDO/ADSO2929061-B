01 - Basic Types

Understanding the fundamental types in TypeScript

String Type

The string type represents textual data in TypeScript.

let message: string = "Hello TypeScript!"; let name: string = 'John Doe'; let template: string = `Welcome ${name}`; console.log(message);

Number Type

TypeScript uses the number type for all numeric values, including integers, floats, hexadecimal, binary, and octal.

let decimal: number = 42; let float: number = 3.14; let hex: number = 0xf00d; let binary: number = 0b1010; let octal: number = 0o744; console.log(decimal + float);

Boolean Type

The boolean type represents logical values: true or false.

let isActive: boolean = true; let isCompleted: boolean = false; if (isActive) { console.log("System is active"); }

Array Types

Arrays can be typed in two ways in TypeScript.

// Method 1: Using square brackets let numbers: number[] = [1, 2, 3, 4, 5]; // Method 2: Using Array generic let fruits: Array<string> = ["apple", "banana", "orange"]; console.log(numbers[0]); console.log(fruits[1]);
